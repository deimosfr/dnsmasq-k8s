# Update Login UX

## Goal
Improve the user experience of the Login feature and global application theme.

## Summary
Refine the Login UI to default to Dark Mode, allow theme toggling on the login page (with specific layout), provide a Logout option in the navigation bar (placed to the right of the theme toggle), and fix an issue where the Option key appears blocked on macOS. Also rename "Welcome Back" to "Dnsmasq K8s".

## Key Changes
- **UX**: Default to Dark Mode.
- **Login**: Add theme toggle to `login.html`.
- **Navbar**: Add "Logout" button.
- **Bugfix**: Ensure standard keyboard input (Option key) is not blocked.
